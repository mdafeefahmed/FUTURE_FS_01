## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/mdafeefahmed/FUTURE_FS_01.git
cd FUTURE_FS_01
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm start
```

---

## 🏗️ Build for Production

```bash
npm run build
```
